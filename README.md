# Customer Analytics Dashboard (Excel + SQL + Power BI)

## 🧾 Overview
This is an end-to-end data analytics project covering the full data pipeline from data preparation in Excel, database design and analysis using SQL, to building an interactive Power BI dashboard.

The goal is to analyze customer behavior, sales performance, and product trends to support data-driven decision making.


## 🔄 Project Workflow
- Data Cleaning & Preparation (Excel)
- Database Design & Modeling (SQL)
- Data Import & Structuring
- SQL Analytical Queries
- Power BI Dashboard Development


## 🛠️ Tools Used
- Excel (Data Cleaning & Preparation)
- SQL (Database Design & Analysis)
- Power BI (Data Visualization & Dashboard)


## 🗄️ Database Structure
The database includes:
- Customer Table
- Product Table
- Deal/Sales Table
- Interaction Data

Relationships were designed to support analytical reporting and business insights.


## 📊 SQL Analysis
Key SQL analyses include:
- Top customers identification
- Total revenue calculation
- Best-selling products analysis
- Customer purchasing behavior analysis

(See analysis_queries.sql for full queries)


## 📈 Dashboard Features
- Interactive filters (Customer, Product, Date)
- Sales performance overview
- Customer segmentation insights
- Revenue trends analysis
- KPI tracking


## 📌 Key Insights
- Identified top contributing customers
- Discovered best-performing products
- Analyzed sales trends over time
- Highlighted customer behavior patterns
